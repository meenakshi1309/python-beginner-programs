## Python Beginner Programs  

This repository contains simple Python programs created for learning and practice.

### Programs Included
- Greeting Program  
- Input Type Checker (int / float / string)  
- Student Result Checker  
- Numeric Password Login  
- Even or Odd using Operators  
- Positive / Negative Number Checker  
- Zero / Non-Zero Detector  
- Last Digit Finder  
- Two-Digit Number Checker  
- Maximum of Two Numbers (Using Operators Only)  

### Purpose
To practice Python fundamentals such as:
- Input handling  
- Conditional statements  
- Operators  
- Basic logic building  

This repository is intended for beginners who are starting their Python learning journey.
